═══════════════════════════════════════════════════════════════════════════════
INTEGRATION PIPELINE
═══════════════════════════════════════════════════════════════════════════════

For each video frame →

  1. OPTICAL FLOW COMPUTATION
     Current frame → calcOpticalFlowFarneback() → Flow field (H×W×2)

  2. FALLBACK POSITION GENERATION
     For each player NOT detected this frame:
       Previous position + flow[y,x] → Fallback position (pixel)
       Confidence *= 0.95

  3. PRIMARY PROJECTION
     Player positions (pixels) → triangulation or homography →
     Field coordinates (meters)

  4. TRAJECTORY VALIDATION
     Each position → velocity check → acceleration check → outlier check
     If invalid: mark as NaN, use fallback instead

  5. KALMAN SMOOTHING
     Each position → KalmanFilterPositionSmoother.smooth() →
     Smoothed position with velocity estimation

  6. HEATMAP ACCUMULATION
     (Smoothed position, team_id, confidence) →
     HeatmapAccumulator.add_frame_with_field_coords() →
     Weighted accumulation in grid
